# AuctaMorph Website

## Setup

1. Install dependencies:
   ```
   npm install
   ```
2. Place media files in `public/`:
   - `logo.png` (your logo)
   - `mach42-bg.mp4` (background video)
   - `starfield.png` (static fallback image)
3. Run in development:
   ```
   npm run dev
   ```
4. Build for production:
   ```
   npm run build
   ```
5. Deploy with Vercel:
   ```
   npm install -g vercel
   vercel --prod
   ```
