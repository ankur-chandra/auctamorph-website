# AuctaMorph Website

This is a Next.js + TailwindCSS website scaffold for AuctaMorph.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Add your logo:
   - Place your `logo.png` file in the `public/` folder.

3. Run locally:
   ```
   npm run dev
   ```

4. Deploy to Vercel:
   Follow the steps in the deployment guide within `README.md`.
